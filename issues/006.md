## What to build

Implement EvaluationEngine in kurma_slm/src/evaluator.py.
Method: .evaluate(blueprint, trace) -> list[DeviationResult]

Three rules (no LLM calls):
1. required step missing from trace -> rule='required_missing', severity='high'
2. prohibited step's tool in trace -> rule='prohibited_action', severity='high'
3. tool in trace not in any blueprint step -> rule='unexpected_tool', severity='medium'

## Acceptance criteria

- [ ] Returns [] when trace satisfies all required steps with no prohibited or unexpected calls
- [ ] required_missing fired for each skipped required step
- [ ] prohibited_action fired when a prohibited step's tool appears in trace
- [ ] unexpected_tool fired for each tool not in any blueprint step
- [ ] Zero LLM calls in this module
- [ ] tests/test_evaluator.py with 4 passing tests
